# 数据集处理Demo

基于 Python Flask 的数据集浏览工具，支持 **PDF 全文检索**（关键词 + 向量语义搜索）和 **视频逐帧像素查看**。采用 **Chinese-CLIP 图文多模态模型**，文字和图像映射到统一向量空间，实现跨模态语义搜索。自动适配任意目录结构，无需修改代码。

## 功能

- **数据集概览** — 自动扫描目录结构，按文件类型分类展示，查看目录树和文件统计
- **文本检索** — 三种搜索模式：
  - **🔤 关键词检索**：jieba 分词 + 倒排索引，精确匹配，含高亮
  - **🧠 语义检索**：FAISS 向量近似搜索（Chinese-CLIP 512维），理解同义词、近义词
  - **⚡ 混合检索**：语义 + 关键词合并排序，兼顾召回率和精确度
- **扫描版PDF** — 自动检测扫描页，渲染为图片并用 CLIP 图像编码器索引，无需 OCR
- **视频抽帧** — 按帧号提取视频画面，支持键盘方向键逐帧浏览
- **像素查看** — 鼠标悬停实时显示 RGB 三通道值和 HEX 色值，点击固定像素详情

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

> **注意**：首次运行会自动下载 Chinese-CLIP 模型（~500MB）。国内环境自动使用 `hf-mirror.com` 镜像。

### 2. 放入数据集

将数据文件放入 `data/` 目录。**子目录名称和层级不限**，程序会自动扫描识别文件类型：

```
data/
├── 书籍/
│   ├── 材料力学.pdf
│   └── 大学化学.pdf
├── 论文/
│   ├── 某论文.pdf
│   └── ...
└── 视频/
    ├── 实验录像.mp4
    └── 演示.wmv
```

> 也可以放到其他目录，启动时用 `--data-dir` 指定即可。

### 3. 启动

```bash
python app.py
```

访问 **http://127.0.0.1:5000**

### 4. 使用

| 标签页 | 说明 |
|--------|------|
| **概览** | 查看数据集目录树、各类型文件统计、关键词索引和 FAISS 向量索引状态 |
| **文本检索** | 输入关键词，下拉选择搜索模式（混合/语义/关键词），结果带匹配方式彩色标签 |
| **视频查看** | 选择视频 → 输入帧号跳转 ← → 键逐帧，鼠标悬停查看 RGB 像素值 |

## 搜索模式详解

| 模式 | 图标 | 原理 | 适用场景 |
|------|------|------|----------|
| 关键词检索 | 🔤 | jieba 分词 + 倒排索引，精确匹配 | 已知精确术语、需要高亮定位 |
| 语义检索 | 🧠 | FAISS + Chinese-CLIP 512维向量，语义近似 | 搜索同义词、跨语言、扫描版PDF |
| 混合检索 | ⚡ | 语义 + 关键词加权合并 | 默认推荐，兼顾精确和召回 |

## 高级用法

### 命令行参数

```bash
python app.py --help
```

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--host` | 绑定地址 | `127.0.0.1` |
| `--port` | 端口号 | `5000` |
| `--data-dir` | 数据集目录 | `./data` |
| `--no-index` | 启动时不构建索引（首次需构建） | 关闭 |
| `--no-vector` | 禁用 FAISS 向量检索 | 关闭 |
| `--debug` | Flask 调试模式 | 关闭 |

### 示例

```bash
# 使用外部数据目录
python app.py --data-dir D:\my-dataset

# 局域网分享
python app.py --host 0.0.0.0 --port 8080

# 仅使用关键词检索（不加载向量模型）
python app.py --no-vector
```

## 工作原理

### 索引阶段（首次启动）

```
PDF文件 → PyMuPDF 提取文字 → 文字正常？→ Chinese-CLIP 文本编码器 → FAISS 向量库
                                    ↓ 乱码
                                PyMuPDF 渲染为图片 → Chinese-CLIP 图像编码器 → FAISS 向量库

视频文件 → OpenCV 采样关键帧 → Chinese-CLIP 图像编码器 → FAISS 向量库

所有文本 → jieba 分词 → 倒排索引（关键词库）
```

### 搜索阶段

```
用户输入 → 同时查询 FAISS 向量库（语义匹配）和倒排索引（关键词匹配）
         → 合并排序 → 返回带匹配模式标签的结果
```

### 缓存机制

- 索引文件缓存在 `cache/` 目录，后续启动秒级加载
- 文件未变更时自动命中缓存，增删文件后自动重建
- FAISS 索引约 10-15MB，关键词索引约 1-2MB
