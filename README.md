# 数据集处理Demo

一个基于 Python Flask 的数据集浏览工具，支持 **PDF 全文检索**和**视频逐帧像素查看**。自动适配任意目录结构，无需修改代码。

## 功能

- **数据集概览** — 自动扫描目录结构，按文件类型分类展示，可查看目录树和文件统计
- **全文检索** — 支持中英文分词搜索所有 PDF 内容，搜索结果高亮，点击查看原文
- **视频抽帧** — 按帧号提取视频画面，支持键盘方向键逐帧浏览
- **像素查看** — 鼠标悬停实时显示 RGB 三通道值和十六进制色值，点击固定像素详情

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 放入数据集

将数据文件放入 `data/` 目录。**子目录名称和层级不限**，程序会自动扫描识别。例如：

```
data/
├── 书籍/
│   ├── 材料力学.pdf
│   └── 大学化学.pdf
├── 论文/
│   ├── 某论文.pdf
│   └── ...
└── 视频/
    ├── 实验录像.mp4
    └── 演示.wmv
```

> 放到其他目录也行，启动时用 `--data-dir` 指定路径即可。

### 3. 启动

```bash
python app.py
```

访问 **http://127.0.0.1:5000**

### 4. 使用

| 标签页 | 操作 |
|--------|------|
| **概览** | 查看数据集目录树、各类型文件数量、索引状态。点「刷新」重新扫描。 |
| **文本检索** | 输入关键词搜索所有 PDF，支持中英文。点搜索结果查看该页原文。 |
| **视频查看** | 下拉选视频 → 输帧号 → 点「跳转」。鼠标在画面上悬停看 RGB，点击固定像素值。键盘 ← → 逐帧切换。 |

## 高级用法

### 命令行参数

```bash
python app.py --help
```

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--host` | 绑定地址 | `127.0.0.1` |
| `--port` | 端口号 | `5000` |
| `--data-dir` | 数据集目录 | `./data` |
| `--no-index` | 启动时不构建搜索索引 | 关闭 |
| `--debug` | Flask 调试模式 | 关闭 |

### 示例

```bash
# 使用外部数据目录
python app.py --data-dir D:\my-dataset

# 局域网内分享给其他人访问
python app.py --host 0.0.0.0 --port 8080
```

## 依赖

| 库 | 用途 |
|----|------|
| Flask | Web 服务 |
| PyMuPDF | PDF 文本提取 |
| OpenCV | 视频帧提取 |
| jieba | 中文分词 |
| Pillow | 图像处理 |
| NumPy | 数组计算 |

## 工作原理

1. **首次启动**：扫描 `data/` 目录 → 提取所有 PDF 文本 → jieba 分词 → 构建倒排索引 → 缓存到 `cache/`
2. **后续启动**：检测文件未变化 → 直接从缓存加载索引 → 秒级启动
3. **数据变更**：增删文件后点「刷新」按钮，或重启程序即自动重建索引
4. **视频帧**：按需提取，不预先生成缩略图，节省磁盘空间
